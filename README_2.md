## Install

```sh
npm install
```

Create a `.env` file in the root directory. Please refer to `src/modules.d.ts` for information about which env vars are required.

## Usage

### **Development**

#### *With Debugging*

Run each command in a single terminal

```sh
npm run build:watch
npm run start:watch
```

#### *Single Compiled Run*

```sh
npm run dev
```

### **Production**

```sh
npm run serve
```

## Testing

**Linting**


```sh
npm run test:lint
```
